Update Migration laravel


php artisan make:migration add_paid_to_users_table --table=users You then need to use the Schema::table() method (as you're accessing an existing table, not creating a new one). And you can add a column like this: public function up() { Schema::table('users', function($table) { $table->integer('paid'); }); } and don't forget to add the rollback option: public function down() { Schema::table('users', function($table) { $table->dropColumn('paid'); }); } Then you can run your migrations: php artisan migrate
